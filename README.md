# Retail Sales Insights Using SQL


## Project Overview

This project focuses on analyzing retail sales data using SQL to uncover customer, product, category, regional, and sales performance insights. The analysis was conducted on a global retail dataset containing order-level transaction records, customer information, product details, and sales metrics.

The objective of this project is to demonstrate how SQL can be used to answer business questions and support data-driven decision-making.

---

## Tools Used

* MySQL Workbench
* SQL

---

## Dataset Overview

The dataset contains:

* Order Information
* Customer Details
* Product Information
* Category and Sub-Category Data
* Regional Sales Data
* Quantity Sold
* Sales Revenue

---

## Business Questions Addressed

### Sales Performance

* What is the total sales revenue?
* How many orders were placed?
* What is the total quantity sold?
* What is the average sales value per order?

### Product Analysis

* Which product categories generate the highest sales?
* Which sub-categories contribute the most revenue?
* What are the top 10 products by sales?

### Customer Analysis

* Who are the top 10 customers by sales?
* Which customer segment contributes the most revenue?

### Regional Analysis

* Which regions generate the highest sales?
* Which states are the top contributors to revenue?

### Operational Analysis

* How are orders distributed across different order priorities?
